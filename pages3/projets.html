Projets

Voici un apperçu des principaux projets que j'ai réalisés. Pour les sites statiques, j'utilise HTML, CSS et JavaScript "vanille" ou encore des librairies comme jQuery ou Bootstrap. J'ai commencé à apprendre React alors j'aurais bientôt de nouvelles réalisations à partager avec vous! Je créé également des sites à partir de maquettes produites dans Photoshop, que j'intègre dans un CMS comme WordPress, qui s'appui sur la base de données MySQL. Enfin, pour les applications web dynamiques, je peux configurer un serveur et concevoir une application en utilisant Node.js et le framework Express. Je peux concevoir une API REST, des gabarits de pages web avec EJS et gérer les autorisations avec Passeport.js. J'utilise MongoDB comme base de données No-SQL, avec le framework Mongoose.


Découvrez plus de projets sur mon compte GitHub ou sur mon compte Codepen.
